Programa de uma Pseudo Clínica Psiquiátrica para alunos do IFSP (CPA IFSP)



Na pasta css contém os programas em css:

bootstrap.css é uma biblioteca pública que peguei na internet para me auxiliar.
Style_login.css é a da página de login
Styles.css é da página dos últimos agendamentos.
Styles.css.map é um arquivo .map



Na pasta images contém imagens jpg que retirei do SUAP, dos alunos, e algumas
imagens svg que encontrei prontas na internet.



Na pasta js temos os programas em javascript:

Dados.js é o javascript utilizado na pagina de login, que faz a verificacao do
nome e senha
index.js é o javascript utilizado na pagina dos agendamentos
Nesse js estão as animações e o ativamento dos botões.

bootstrap.js, jquery.js e popper.js são bibliotecas públicas que auxiliam na troca
de telas pelo botao da agenda em cima, e com as animações como a da agenda, ao
passar o mouse em cima


Login.html é a página html do Login
Index.html é a página html da página de agendamentos
Cadastro.html é uma página em html que pretendo implementar futuramente para fazer o
cadastro dos médicos e pacientes (não foi apresentada em aula, está incompleta)
